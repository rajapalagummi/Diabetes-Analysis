# Comprehensive Analysis & Visualisation of Progression of Diabetes

CDC BRFSS Diabetes Health Indicators — 70,000+ US respondents.
6 classification algorithms compared. 19 output files generated automatically.

## Setup

```bash
python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python3 src/diabetes_analysis.py
```

Outputs saved to /outputs folder automatically.
